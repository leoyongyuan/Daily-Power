# 生产与开发配置

在生产和开发时的配置是由很多区别的，生产时，我们需要压缩代码并且缓存，不需要引入debug功能。

## 公共路径

公共路劲可以指定我们的应用程序中所有资源的基础路径。

使用
```Javascript
module.exports = {
    output: {
        publicPath: 'http://localhost:8080/',       // 设定公共路径
    },
}
```

## 环境配置

webpack 提供一个 --env的参数，可以让我们传入环境变量。例如 --env production，传入一个生产模式。

在使用这个模式时，我们需要把module中修改成一个函数。
```Javascript
module.exports = (env) => {
    return {
      entry: {
        ...
      },
      
      output: {
        ...
      }
    }
}

```
